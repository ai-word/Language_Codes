# multipart

Multipart's `Getting Started` guide for Actix web

## Usage

### server

```bash
cd examples/multipart
cargo run (or ``cargo watch -x run``)
# Started http server: 127.0.0.1:8080
```

### client

- ``pip install aiohttp``
- ``python client.py``
- you must see in server console multipart fields

if ubuntu :

- ``pip3 install aiohttp``
- ``python3 client.py``
